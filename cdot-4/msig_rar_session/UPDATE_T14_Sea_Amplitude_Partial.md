## 2. Why $\rho_\text{bg}\sim\rho_\text{crit}$ — Motivated, Not Derived

T14 identifies the connecton energy $\hbar H_0$ as the holographic IR-cutoff dark-energy
quantum. That identification naturally makes the connecton sea density of order the
dark-energy / critical density: $\rho_\text{bg}\sim\rho_\Lambda\sim\rho_\text{crit}$. So the
model *does* motivate $\rho_\text{bg}\sim\rho_\text{crit}$ — but only to **order unity**,
since $\Omega_\Lambda = 0.68$, not 1. That residual $\Omega$-factor **is** the leftover ~2
in $S$. The forward estimate wants $\rho_\text{bg}\approx0.46\,\rho_\text{crit}$ to hit $a_0$
exactly, which has no clean standard-parameter identification.

Across four sessions the $a_0$ coefficient has been attacked three ways — single-wavelength,
mean-gradient, and this amplitude route. **Every attempt resolves the same way: order unity,
dimensionally natural, not precisely pinned.** That consistency is itself informative:

  indistinguishability; the $q=2$ spectrum slope; horizon-domination of the floor; the $1/r$
  profile and its constant acceleration. The **6** in $cH_0/6$ is fixed by horizon geometry

- **Not derived:** the absolute O(1) factor $S$. It is dimensionally forced to be $\sim1$ by
  the connecton = dark-energy identification, consistent with data, but its precise value is
  tied to $\rho_\text{bg}/\rho_\text{crit}$ — an O(1) ratio the model motivates but does not

**What would be self-deception:** stacking three unpinned O(1) factors (sea = critical,
column geometry $\Sigma=\rho R_0$, profile factor $1/2$) to land on $S=1$. That is
curve-fitting dressed as derivation, and this update explicitly does **not** claim it. The
forward estimate in fact overshoots by ~2×.

**The defensible statement:** the model predicts
$$a_0 = \frac{cH_0}{6}\,S,\qquad S = O(1)\ \text{dimensionally forced by the connecton–dark-energy identification},$$
consistent with the measured $a_0$, with the exact O(1) set by $\rho_\text{bg}/\rho_\text{crit}$.
This is the honest ceiling of the current derivation.
